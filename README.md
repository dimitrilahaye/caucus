# cré-impro

# Get started

Installation des dépendances:

```bash
npm i
```

Lancer le serveur de dev (local + réseau):

```bash
npm run dev -- --host
```

Construire pour la prod:

```bash
npm run build
```

Prévisualiser le build localement:

```bash
npm run preview
```

Déployer sur GitHub Pages (branche `gh-pages`):

```bash
npm run deploy
```
